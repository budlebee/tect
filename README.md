의견과 참여 늘 환영합니다.

현재(20.11.05) dev 브랜치는 gh-pages 용 페이지 개발 브랜치이고,
CRUD 기능이 포함된 풀스택 페이지는 다른 브랜치에서 개발하고 있습니다.

master 브랜치는 개발용 브랜치들에 아무런 버그가 없으면 간간히 merge 하려고 냅두고 있습니다.
